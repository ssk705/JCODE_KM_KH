public class DoublyLinkedList {
     class Node {
        int data;
        Node next;
        Node prev;
    }
   public void insertAfter(Node prevNode, int data) {
    Node newNode = new Node(data);
    newNode.next = prevNode.next;  
    prevNode.next = newNode;      
} 
}

KM:The new node's `prev` pointer isn't set to `prevNode`, and the node *after* `newNode` doesn't have its `prev` updated to point back.
